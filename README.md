# h2-demo

HTTP/2 Demo 

Check it [here](https://http2-1035.appspot.com) 

Same code base on HTTP/1.1 [here](http://www.http2-1035.appspot.com/)

### Development

```js
git clone git@github.com:vigneshshanmugam/h2-demo.git
npm install
```

##### HTTP/2

```js
npm run http2

go to https://localhost:3000
```

##### HTTP/1.1

```js
npm run start

go to https://localhost:3000
```
